# TOOLS.md - 本地笔记

技能定义了你的工具如何工作。这个文件是用于你的细节——你的设置是独一无二的。

## 这里放什么

例子：

```
### 摄像头
- living-room → 主区域，180度广角
- front-door → 入口， motion-triggered

### SSH
- home-server → 192.168.1.100, user: admin

### TTS
- 首选语音："Nova"（温暖，略英国）
- 默认扬声器：Kitchen HomePod
```

## 为什么分开？

技能是共享的。你的设置是你的。保持它们分开意味着你可以更新技能而不丢失你的笔记，分享技能而不泄露你的基础设施。
